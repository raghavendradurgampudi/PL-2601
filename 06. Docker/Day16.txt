Dockerfile
Dockerfile will have instructions to build a docker image
Dockerfile uses DSL (Domain Specific Language) key-words
All the instructions in the Dockerfile will be processed from top to bottom
Any Dockerfile keyword must be written in CAPITAL letters
Every Dockerfile keyword must be written in new line

FROM
It indicates the base image to run our application
It is used to represent the base image to create our own image
We can write multiple FROM instructions in a single Dockerfile
FROM <ImageName>

RUN
It is used to execute the commands - the commands gets executes while creating the image
We can write multiple RUN instructions in a single Dockerfile
RUN <Command>

CMD
It is also used to execute the commands - the commands gets executes while creating the container
We can write multiple CMD instructions in a single Dockerfile, but the last CMD instruction in the Dockerfile will only be processed
CMD <Command>
